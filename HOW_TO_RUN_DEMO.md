If you want to show this project to a recruiter, colleague, or friend, follow these 3 simple steps:

### 1. Prerequisites
Ensure you have **Node.js** installed on your computer. You can check by typing `node -v` in your terminal.

### 2. Open the Project
1. Open your terminal (or Command Prompt).
2. Navigate to this folder:
   ```bash
   cd "C:\AI JOB SEEKING\job-portal"
   ```

### 3. Start the Demo
Type the following command:
```bash
npm run dev
```

### 4. View the project
Open your browser and go to:
[**http://localhost:3000**](http://localhost:3000)

- **Admin Dashboard**: Go to `/admin/dashboard` to show the management side.
